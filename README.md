# Backgroun_Remover_AK
Can remove backgrounds and can fill them with colors and runs locally.

Benefits:
*Runs locally
*Can fix edges with granular controls
*Zoom in and out to fix easily

Uses rembg [https://github.com/danielgatis/rembg]
